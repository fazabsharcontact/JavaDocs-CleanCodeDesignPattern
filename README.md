# Proyek Dokumentasi Java — Calculator

## Deskripsi Proyek
Proyek ini merupakan tugas pembuatan **dokumentasi kode Java menggunakan JavaDoc**.  
Program sederhana ini berisi kelas `Calculator` yang memiliki dua operasi dasar:
- **Penjumlahan** (`tambah`)
- **Pengurangan** (`kurang`)

Tujuan utama proyek ini adalah untuk mempraktikkan pembuatan dokumentasi yang jelas dan terstruktur menggunakan JavaDoc, sehingga dapat mempermudah proses **pemeliharaan, pembaruan, dan kolaborasi tim** dalam pengembangan perangkat lunak.

---

## Informasi Teknis
- **Bahasa Pemrograman:** Java  
- **JDK yang digunakan:** Java SE Development Kit (JDK) 17  
- **IDE:** IntelliJ IDEA Community Edition  
- **Tool Dokumentasi:** JavaDoc (bawaan JDK 17)
